flexCarousel.js is a jQuery plugin which allows developers to create customisable and accessible carousel components for their websites by utilizing Flexbox browser technology. This plugin aims to stay on the edge of browser technology and will constantly be maintained as a reliable plugin that can be easily adapted to your needs whilst being compatible on any device.
