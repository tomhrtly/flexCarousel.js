It's very easy to customise your carousel by changing the options given to you. All you have to do is invoke the flexCarousel function and pass an object with the options you wish to change from the default value.

{% highlight html %} {% endhighlight %}

For a full list of all the options available, read the table below. There is also an example link so that you can see a live demo of what each option does.

Name Type Default Description Example
arrows Boolean true Determines whether a previous and next arrow should be shown on either side of the slides so that the user can change the active slide.
arrowsOverlay Boolean true arrows must be set to true for this option to work. Determines whether the arrows overlay onto the active slide or have their own wrapping elements.
autoplay Boolean false Determines whether the active slide changes to the next slide on a time delay.
autoplaySpeed Integer 5000(ms) autoplay must be set to true for this option to work. Determines how long the active slide will stay active for before transitioning to the next slide.
height String null Defines whether a fixed height is to be added to the carousel selector element.
nextArrow String '<svg>...</svg>' Defines the HTML code for the element which changes the active slide to the next slide when clicked on. Out of the box, we use Font Awesome icons.
prevArrow String '<svg>...</svg>' Defines the HTML code for the element which changes the active slide to the previous slide when clicked on. Out of the box, we use Font Awesome icons.
slidesVisible Integer 1 Defines how many slides are visible at any given time.
transition String 'slide' Defines which transition should be used when changing the active slide. Current transitions available are: 'none' and 'slide'.
