Box-sizing

The majority of developers will already use box-sizing: border-box; for their web pages, but in case the box-sizing property isn't already set to the border-box value, we have done this for you to ensure correct sizing of the carousel component. You shouldn't need to change this value.

HTML structure

It's very easy to create a carousel with flexCarousel.js, the HTML you have to write could not be simpler. First you create a parent div tag which acts as a selector for the carousel component. We recommend using a class name that does not use any other JavaScript or CSS.

Then you create as many slides for that carousel as you need by using the div tag. So for three slides, you would create three child div elements inside the parent element.

For example, if you were making a carousel for employees on your web page, it is recommended to use .employees-js for your selector and all of the CSS styles would be for .employees.
