flexCarousel.js is a JavaScript plugin which allows developers to create customisable and accessible carousel components for their websites by utilizing Flexbox browser technology. This plugin aims to stay on the edge of browser technology and will constantly be maintained as a reliable plugin that can be easily adapted to your needs whilst being compatible on any device.

It's very easy to get setup and start customising your carousels for your web page, you can find several examples and detailed information to ensure your development experience is as smooth as possible.
